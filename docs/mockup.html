One-Way ANOVA

Tests whether the means of three or more independent groups differ significantly. Assumes each group is normally distributed with equal variance (homoscedasticity).

When to use: You have one categorical independent variable with 3+ levels and one continuous outcome. Groups are independent (different subjects).

Assumptions:

  • Normality within each group (check with Shapiro-Wilk)
  • Equal variances (Levene's test)
  • Independent observations

If assumptions are violated: Use Kruskal-Wallis (non-parametric alternative).

After a significant result: Run a post-hoc test (Tukey HSD, Bonferroni, or Dunnett vs control) to determine which specific pairs differ.

Reference: Zar, J.H. (2010). Biostatistical Analysis, 5th ed. Prentice Hall.
